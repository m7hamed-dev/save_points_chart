# save_points_chart example

Demo app for [save_points_chart](https://pub.dev/packages/save_points_chart).

## Run locally

```bash
cd example
flutter pub get
flutter run
```

## Publishing

**Do not** run `dart pub publish` from this directory. This folder is an example app (`publish_to: none`), not the library package.

Publish the library from the repository root:

```bash
cd ..   # save_points_chart package root
dart pub publish
```
